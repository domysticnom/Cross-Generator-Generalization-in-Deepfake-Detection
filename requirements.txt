# ---------------------------------------------------------------------------
# PyTorch is installed SEPARATELY from the cu128 wheel index, NOT from PyPI:
#
#   pip install torch==2.9.* torchvision==0.24.* torchaudio==2.9.* \
#       --index-url https://download.pytorch.org/whl/cu128
#
# The default PyPI torch wheel ships without sm_120 kernels and will fail on
# RTX 50-series (Blackwell) GPUs. The cu128 wheel also runs on older cards
# (e.g. Tesla P100, sm_60), so one install serves the whole team.
# ---------------------------------------------------------------------------
